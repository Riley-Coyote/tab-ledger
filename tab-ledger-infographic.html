Browser Tabs
Chrome DevTools Protocol + Session Files
Captures the current state of all open browser tabs via CDP on ports 9222–9224, with a fallback to parsing session storage files directly.
CDP 9222–9224 20 categories
  • URL regex categorization
  • Stale tab detection (expired OAuth, dead localhost)
  • History DB cross-reference for visit counts
  • Session file fallback when CDP unavailable

snapshot.py
Tab Capture Engine
Captures browser state through a multi-step pipeline with intelligent fallbacks and enrichment from the browser's History database.
  • Try CDP on ports 9222, 9223, 9224
  • Fallback: parse session storage files
  • Cross-reference with History DB
  • Apply URL regex categorization (20 cats)
  • Check stale: expired OAuth, dead localhost
  • Save snapshot + individual tab records
CDP primary session fallback
